# Market Analyzer

AI-powered market insight engine with multi-agent analysis.

## Features

- Autonomous market opportunity discovery
- Multi-agent analysis (macro, sector, technical, risk)
- Insight conversations and follow-up research
- Statistical signals and pattern tracking
- Outcome tracking with learning loop

## Prerequisites

- Python 3.11+
- Node.js 18+
- uv (Python package manager)
- npm

## Installation

### Backend

```bash
cd backend
uv sync
```

### Frontend

```bash
cd frontend
npm install
```

## Running

### Backend

```bash
cd backend
uv run uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm run dev
```

## API

- Backend: http://localhost:8000
- Frontend: http://localhost:3000
- API Docs: http://localhost:8000/docs
